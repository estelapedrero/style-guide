# Mr.Nash Style Guide
Style Guide. It set the basis to create style rules of any project

# Installing the project

### Clone the repository
```
$ git clone https://github.com/estelapedrero/style-guide.git
```

### Access the project
```
$ cd style-guide
```

### Install the dependencies
```
$ npm install
```

### Run the application
```
$ npm start
```

### Lint test
```
$ npm run lint:js
